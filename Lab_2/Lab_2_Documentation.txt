1. Install Apache Maven
	https://www.youtube.com/watch?v=XEphzGQz-nI
2. Run the unit test
	a. check fix done in AppTest.java file
		The unit test initially failed due to a line-ending mismatch on Windows (\r\n vs \n). 
		The test was updated to use System.lineSeparator() to ensure cross-platform compatibility.
	b. From the location Lab_2/build2-main, run -> mvn test
3. Build the app
	run -> mvn package
4.Verify artifact
	run -> ls target
	expected -> hello-ivolve-1.0-SNAPSHOT.jar
5. Run the app
	run -> java -jar target/hello-ivolve-1.0-SNAPSHOT.jar
	expected -> Hello iVolve Trainee
